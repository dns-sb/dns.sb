# How to change DNS settings on macOS

<StepContainer>

### Open System Settings

Go to `System Settings` and select `Network`:

[![macOS System Preferences Network settings](/images/guide/macos/macos-system-preferences-network.webp)](/images/guide/macos/macos-system-preferences-network.webp)

Select your connection type (`Wi-Fi` or `Ethernet`).

### Configure DNS Servers

Click `Details...` on your active connection, then select the `DNS` tab.

Remove any existing DNS servers and add the following:

```bash filename="IPv4"
185.222.222.222
```

```bash filename="IPv4"
45.11.45.11
```

```bash filename="IPv6"
2a09::
```

```bash filename="IPv6"
2a11::
```

[![macOS DNS server settings](/images/guide/macos/macos-dns-server-settings.webp)](/images/guide/macos/macos-dns-server-settings.webp)

### Save Changes

Click `OK` to apply the new DNS settings.

</StepContainer>
