# How to change DNS settings on Windows

<Callout type="info">
For a simpler setup, consider using [Private DNS (DNS over HTTPS)](/doh/windows/) which is available on Windows 11 and later.
</Callout>

## Windows 11 / Windows Server 2022

<StepContainer>

### Open Network Settings

Right-click the network icon in the taskbar:

[![Windows 11 taskbar network icon](/images/guide/windows/win11-taskbar-network-icon.webp)](/images/guide/windows/win11-taskbar-network-icon.webp)

Select `Network and Internet settings`:

[![Windows 11 Network and Internet settings menu](/images/guide/windows/win11-network-internet-settings-menu.webp)](/images/guide/windows/win11-network-internet-settings-menu.webp)

Click `Ethernet` or `Wi-Fi` depending on your connection type:

[![Windows 11 Ethernet or Wi-Fi selection](/images/guide/windows/win11-ethernet-wifi-selection.webp)](/images/guide/windows/win11-ethernet-wifi-selection.webp)

### Configure DNS Servers

In the `DNS server assignment` section, click `Edit`:

[![Windows 11 DNS server assignment edit button](/images/guide/windows/win11-dns-server-edit-button.webp)](/images/guide/windows/win11-dns-server-edit-button.webp)

Change from `Automatic (DHCP)` to `Manual`:

[![Windows 11 DNS manual configuration dialog](/images/guide/windows/win11-dns-manual-configuration.webp)](/images/guide/windows/win11-dns-manual-configuration.webp)

Enter the following DNS servers:

**IPv4**

| Setting | Value |
|---------|-------|
| Preferred DNS | `185.222.222.222` |
| Alternate DNS | `45.11.45.11` |

[![Windows 11 DNS IPv4 settings](/images/guide/windows/win11-dns-ipv4-settings.webp)](/images/guide/windows/win11-dns-ipv4-settings.webp)

**IPv6**

| Setting | Value |
|---------|-------|
| Preferred DNS | `2a09::` |
| Alternate DNS | `2a11::` |

[![Windows 11 DNS IPv6 settings](/images/guide/windows/win11-dns-ipv6-settings.webp)](/images/guide/windows/win11-dns-ipv6-settings.webp)

Click `Save` to apply the changes.

</StepContainer>

## Windows 10 / Windows Server 2019

<StepContainer>

### Open Network Settings

Right-click the network icon in the taskbar:

[![Windows 10 taskbar network icon](/images/guide/windows/win10-taskbar-network-icon.webp)](/images/guide/windows/win10-taskbar-network-icon.webp)

Select `Open Network & Internet settings`:

[![Windows 10 Network and Internet settings menu](/images/guide/windows/win10-network-internet-settings-menu.webp)](/images/guide/windows/win10-network-internet-settings-menu.webp)

Click `Change adapter options`:

[![Windows 10 change adapter options](/images/guide/windows/win10-change-adapter-options.webp)](/images/guide/windows/win10-change-adapter-options.webp)

Right-click your active network adapter and select `Properties`:

[![Windows 10 adapter properties menu](/images/guide/windows/win10-adapter-properties-menu.webp)](/images/guide/windows/win10-adapter-properties-menu.webp)

### Configure IPv4 DNS

Select `Internet Protocol Version 4 (TCP/IPv4)` and click `Properties`:

[![Windows 10 TCP/IPv4 properties](/images/guide/windows/win10-tcpip-v4-properties.webp)](/images/guide/windows/win10-tcpip-v4-properties.webp)

Select `Use the following DNS server addresses` and enter:

| Setting | Value |
|---------|-------|
| Preferred DNS server | `185.222.222.222` |
| Alternate DNS server | `45.11.45.11` |

[![Windows 10 DNS IPv4 settings](/images/guide/windows/win10-dns-ipv4-settings.webp)](/images/guide/windows/win10-dns-ipv4-settings.webp)

Click `OK` to save.

### Configure IPv6 DNS

Select `Internet Protocol Version 6 (TCP/IPv6)` and click `Properties`:

[![Windows 10 TCP/IPv6 properties](/images/guide/windows/win10-tcpip-v6-properties.webp)](/images/guide/windows/win10-tcpip-v6-properties.webp)

Select `Use the following DNS server addresses` and enter:

| Setting | Value |
|---------|-------|
| Preferred DNS server | `2a09::` |
| Alternate DNS server | `2a11::` |

[![Windows 10 DNS IPv6 settings](/images/guide/windows/win10-dns-ipv6-settings.webp)](/images/guide/windows/win10-dns-ipv6-settings.webp)

Click `OK` to save.

</StepContainer>
