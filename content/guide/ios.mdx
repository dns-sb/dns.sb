# How to change DNS settings on iOS

<StepContainer>

### Go to `Settings` and select `Wi-Fi`

![5k0i.png](https://s3.image.hosting/2021/07/02/5k0i.png)

### Tap the `i` button to the right of your connected Wi-Fi network

![5w3l.png](https://s3.image.hosting/2021/07/02/5w3l.png)

### Scroll down and find `Configure DNS`

![KBrK.png](https://s3.image.hosting/2021/07/02/KBrK.png)

### Select `Manual`, remove old DNS servers and then add DNS.SB's servers

```
185.222.222.222
```

```
45.11.45.11
```

```
2a09::
```

```
2a11::
```

![KYhU.png](https://s3.image.hosting/2021/07/02/KYhU.png)

### Click on `Save`

</StepContainer>
