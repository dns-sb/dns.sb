# How to change DNS settings on iOS

<StepContainer>

### Open Wi-Fi Settings

Go to `Settings` and tap `Wi-Fi`:

[![iOS WiFi settings](/images/guide/ios/ios-wifi-settings.webp)](/images/guide/ios/ios-wifi-settings.webp)

### Access Network Details

Tap the `â“˜` button next to your connected Wi-Fi network:

[![iOS WiFi network details](/images/guide/ios/ios-wifi-network-details.webp)](/images/guide/ios/ios-wifi-network-details.webp)

### Open DNS Configuration

Scroll down and tap `Configure DNS`:

[![iOS configure DNS settings](/images/guide/ios/ios-configure-dns.webp)](/images/guide/ios/ios-configure-dns.webp)

### Configure DNS Servers

1. Select `Manual`
2. Remove existing DNS servers
3. Add the following servers:

```bash
185.222.222.222
```

```bash
45.11.45.11
```

```bash
2a09::
```

```bash
2a11::
```

[![iOS DNS server settings](/images/guide/ios/ios-dns-server-settings.webp)](/images/guide/ios/ios-dns-server-settings.webp)

4. Tap `Save`
