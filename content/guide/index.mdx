# How to use DNS.SB

## DNS.SB Name Servers

### IPv4

```bash filename="IPv4"
185.222.222.222
```

```bash filename="IPv4"
45.11.45.11
```

### IPv6

```bash filename="IPv6"
2a09::
```

```bash filename="IPv6"
2a11::
```

Some older devices may require the full IPv6 address:

```bash filename="IPv6"
2a09:0000:0000:0000:0000:0000:0000:0000
```

```bash filename="IPv6"
2a11:0000:0000:0000:0000:0000:0000:0000
```
