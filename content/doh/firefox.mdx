# How to set DoH on Firefox

<StepContainer>

### Update Firefox

Ensure you have the latest version of Firefox installed. You can download it from the [official website](https://www.mozilla.org/firefox/all/).

### Open Settings

Enter `about:preferences` in the address bar and search for `DNS over HTTPS`.

### Configure DNS over HTTPS

On the `Enable DNS over HTTPS using` section:

1. Select `Increased Protection`
2. Set the provider to `Custom`
3. Enter the following URL:

```bash filename="URL"
https://doh.dns.sb/dns-query
```
