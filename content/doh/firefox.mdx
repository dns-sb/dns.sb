# How to set DoH on Firefox

<StepContainer>

### Upgrade your Firefox to latest version

Make sure you have installed the latest version of Firefox, or you can download it [here](https://www.mozilla.org/en-US/firefox/all/).

### Go to Settings

Input `about:preferences` in address bar and search `DNS over HTTPS` on search bar.


### Set DNS.SB DoH Server

On the `Enable DNS over HTTPS using` page, choose `Increased Protection`, select provider as `Custom`, and enter:

```
https://doh.dns.sb/dns-query
```
