# How to set DoH on Microsoft Edge

This guide is for Microsoft Edge and most Chromium-based browsers.

<StepContainer>

### Update Microsoft Edge

Ensure you have the latest version of Microsoft Edge installed. You can download it from the [official website](https://www.microsoft.com/edge/).

### Open Security Settings

Enter `edge://settings/privacy` in the address bar and scroll down to the `Security` section.

### Configure DNS over HTTPS

In the `Use secure DNS` section:

1. Enable the toggle
2. Select `Choose a service provider`
3. Enter the following URL:

```bash filename="URL"
https://doh.dns.sb/dns-query
```
