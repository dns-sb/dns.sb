# How to set DoH on Chrome

This guide is for Google Chrome and most Chromium-based browsers.

<StepContainer>

### Update Chrome

Ensure you have the latest version of Chrome installed. You can download it from the [official website](https://www.google.com/chrome/).

### Open Security Settings

Enter `chrome://settings/security` in the address bar.

### Configure DNS over HTTPS

In the `Advanced` section:

1. Enable `Use secure DNS`
2. Select `Add custom DNS service provider`
3. Enter the following URL:

```bash filename="URL"
https://doh.dns.sb/dns-query
```
