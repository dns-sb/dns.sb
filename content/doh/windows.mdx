# How to set DNS over HTTPS on Windows

This guide covers DNS over HTTPS configuration for Windows 10 and Windows 11.

## How to set DoH on Windows 11

<StepContainer>

### Open Network Settings

Right-click the network icon in the taskbar:

[![Windows 11 taskbar network icon](/images/doh/windows/win11-taskbar-network-icon.webp)](/images/doh/windows/win11-taskbar-network-icon.webp)

Select `Network and Internet settings`:

[![Windows 11 Network and Internet settings](/images/doh/windows/win11-network-internet-settings.webp)](/images/doh/windows/win11-network-internet-settings.webp)

### Configure Your Connection

Click `Ethernet` (or `Wi-Fi` if using a wireless connection):

[![Windows 11 Ethernet settings](/images/doh/windows/win11-ethernet-settings.webp)](/images/doh/windows/win11-ethernet-settings.webp)

In the `DNS server assignment` section, click `Edit`:

[![Windows 11 DNS server assignment edit](/images/doh/windows/win11-dns-server-assignment-edit.webp)](/images/doh/windows/win11-dns-server-assignment-edit.webp)

### Enable DNS over HTTPS

Configure the following settings:

1. Enter your preferred DNS server IP address
2. Set `DNS over HTTPS` to `On (manual template)`
3. Enter `https://doh.dns.sb/dns-query` as the `DNS over HTTPS template` (or choose another [unicast server](/doh/))
4. Click `Save`

[![Windows 11 DoH encryption settings](/images/doh/windows/win11-doh-encryption-settings.webp)](/images/doh/windows/win11-doh-encryption-settings.webp)

</StepContainer>

## How to set DoH on Windows 10

Windows 10 does not have native DNS over HTTPS support. You can use a third-party application like YogaDNS.

<StepContainer>

### Install YogaDNS

Download and install the latest version from the [official website](https://yogadns.com/download/).

### Launch with Empty Configuration

After installation, launch YogaDNS. Start with an empty configuration:

[![YogaDNS empty configuration](/images/doh/windows/yogadns-empty-configuration.webp)](/images/doh/windows/yogadns-empty-configuration.webp)

### Add DNS.SB DoH Server

Click `DNS Servers`, then click `Add...`

[![YogaDNS DNS servers menu](/images/doh/windows/yogadns-dns-servers-menu.webp)](/images/doh/windows/yogadns-dns-servers-menu.webp)

Configure the following settings:

- Set `Type` to `DNS over HTTPS`
- Set `IP address and optional port` to `185.222.222.222`
- Set `DNS over HTTPS options` URL to `https://doh.dns.sb/dns-query`

[![YogaDNS DoH server settings](/images/doh/windows/yogadns-doh-server-settings.webp)](/images/doh/windows/yogadns-doh-server-settings.webp)

Click `OK` to save.

</StepContainer>
