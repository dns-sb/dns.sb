# How to set DoH on Vivaldi

<StepContainer>

### Upgrade your Vivaldi to latest version

Make sure you have installed the latest version of Vivaldi, or you can download it [here](https://vivaldi.com/download/).

### Go to Settings

Input `vivaldi:settings/network/` in address bar or use `ctrl` + `F12` to open settings page.

### Set DNS.SB DoH Server

In the `Network` section (or you can search for `DNS Over HTTPS`), enable `DNS Over HTTPS`, choose `Custom DNS Provider`, and enter:

```
https://doh.dns.sb/dns-query
```
