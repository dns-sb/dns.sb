# How to set DoH on Vivaldi

<StepContainer>

### Update Vivaldi

Ensure you have the latest version of Vivaldi installed. You can download it from the [official website](https://vivaldi.com/download/).

### Open Network Settings

Enter `vivaldi://settings/privacy/` in the address bar, or press `Ctrl` + `F12` to open settings, then navigate to `Privacy and Security`.

### Configure DNS over HTTPS

In the `DNS over HTTPS` section:

1. Enable `DNS over HTTPS`
2. Select `Custom`
3. Enter the following URL:

```bash filename="URL"
https://doh.dns.sb/dns-query
```
