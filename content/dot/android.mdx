# How to set DNS over TLS on Android

## Android 11+

Android 11 and later versions have built-in support for DNS over TLS via the Private DNS feature.

<StepContainer>

### Open Network Settings

Go to `Settings` and open `Network & internet`:

[![Android settings menu](/images/dot/android/android-settings-menu.webp)](/images/dot/android/android-settings-menu.webp)

### Access Advanced Options

Tap `Advanced`:

[![Android Network and Internet settings](/images/dot/android/android-network-internet-settings.webp)](/images/dot/android/android-network-internet-settings.webp)

### Open Private DNS Settings

Tap `Private DNS`:

[![Android Private DNS settings](/images/dot/android/android-private-dns-settings.webp)](/images/dot/android/android-private-dns-settings.webp)

### Configure DNS.SB

1. Select `Private DNS provider hostname`
2. Enter the following hostname:

```txt
dot.sb
```

[![Android Private DNS hostname input](/images/dot/android/android-private-dns-hostname.webp)](/images/dot/android/android-private-dns-hostname.webp)

3. Tap `Save`

</StepContainer>

## Alternative Apps

### Nebulo

[Nebulo](https://play.google.com/store/apps/details?id=com.frostnerd.smokescreen) is a free, open-source, no-root, lightweight DNS client for Android that supports DNS-over-HTTPS, DNS-over-TLS, and DNS-over-QUIC protocols with emphasis on privacy and security.

- [Google Play](https://play.google.com/store/apps/details?id=com.frostnerd.smokescreen)
- [Source Code](https://git.frostnerd.com/PublicAndroidApps/smokescreen)

### Intra

[Intra](https://play.google.com/store/apps/details?id=app.intra) protects you from DNS manipulation, a type of cyber attack used to block access to news sites, social media platforms, and messaging apps.

- [Google Play](https://play.google.com/store/apps/details?id=app.intra)
- [Source Code](https://github.com/Jigsaw-Code/intra)
