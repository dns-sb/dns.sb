# How to set DoH on Chrome

This guide is for Google Chrome, also works with most Chromium based browsers.

<StepContainer>

### Upgrade your Chrome to latest version

Make sure you have installed the latest version of Chrome, or you can download it [here](https://www.google.com/chrome/).

### Go to Settings

Input `chrome://settings/security` in address bar.

### Set DNS.SB DoH Server

In the `Advanced` section, enable `Use secure DNS`, choose `Add custom DNS service provider`, and enter:

```
https://doh.dns.sb/dns-query
```
