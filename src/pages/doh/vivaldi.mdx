# How to set DoH on Vivaldi

<StepContainer>

### Upgrade your Vivaldi to latest version

Make sure you have installed latest Vivaldi, or you can download [here](https://vivaldi.com/download/).

### Go to Settings

Input `vivaldi:settings/network/` in address bar or use `ctrl` + `F12` to open settings page.

### Set DNS.SB DoH Server

On `Network` section, or you can search `DNS Over HTTPS`, then enable `DNS Over HTTPS`, choose `Custom DNS Provider` and input:

```
https://doh.dns.sb/dns-query
```
