# How to set DoH on Microsoft Edge

This guide is for Microsoft Edge, also works with most Chromium based browsers.

<StepContainer>

### Upgrade your Microsoft Edge to latest version

Make sure you have installed the latest version of Microsoft Edge, or you can download it [here](https://www.microsoft.com/edge/).

### Go to Settings

Input `edge://settings/privacy/security` in address bar.

### Set DNS.SB DoH Server

In the `Use secure DNS` section, enable it, choose `Choose a service provider`, and enter:

```
https://doh.dns.sb/dns-query
```
